Practice with Functions and Events

Learn to about functions and events!

Every HTML Element can have an ID. But it must be unique on the page.

JavaScript can "hook" into HTML elements and change their properties or behaviors. They need to know the ID of the element they want to hook into.

In JavaScript, we can define functions that can be called later. The function definition is stored in memory, and the function can be called later by using the function name.

That function can be called when the user clicks a button. We call this "attaching an event listener" to the element. We can use the helper functions in the helpers.js file to do this.

Your challenge is onEvent to attach an event listener to each button and use setProperty to change the color of the text of the paragraph above it.

Example: change a CSS property of an element

        setProperty("header1", "color", "red");
        setProperty("header1", "border-style", "solid");
        setProperty("header1", "border-width", "1px");
        setProperty("header1", "border-color", "red");
        setProperty("header1", "border-radius", "5px");
        setProperty("header1", "padding", "10px");
        setProperty("header1", "margin", "10px");
        setProperty("header1", "font-size", "20px");

Example: change the CSS property of an element when the button is clicked

        onEvent("button1", "click", function() {
            setProperty("paragraph1", "color", "red");
        });

Example: Change the text of an element

        setProperty("header1", "textContent", "Hello World");
